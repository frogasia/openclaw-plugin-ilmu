Initial release. Registers Ilmu as an OpenClaw LLM provider with two models (`nemo-super`, `ilmu-nemo-nano`), 256k context window, OpenAI-compatible API at `https://api.ilmu.ai/v1`, activated via `ILMU_API_KEY`. Ports the work originally proposed in [frogasia/openclaw#1](https://github.com/frogasia/openclaw/pull/1) to a standalone Clawhub plugin.
